# loreth.co

Personal portfolio site for Andre Loreth. Built with [Astro](https://astro.build)
and [Tailwind CSS v4](https://tailwindcss.com).

## Development

```bash
yarn install
yarn dev
yarn build
```

Node version pinned in `.tool-versions`.

## Content

All copy lives in `src/data/site.ts`.

## Deployment

Auto-deploys to GitHub Pages on push to `main` via
`.github/workflows/deploy.yml`.
